# 🎣 Nusantara Fishing Adventure

Nusantara Fishing Adventure — game mancing Indonesia berbasis browser, terinspirasi dari Hooked on Creatures of the Deep (N-Gage).

## Fitur
- 5 lokasi mancing: Danau Toba, Raja Ampat, Sungai Mahakam, Laut Jawa, Danau Matano
- 18+ spesies ikan dengan rarity Common / Rare / Epic / Legendary
- NPC lokal tiap daerah dengan dialog, tips, dan cerita lokal
- 15 quest tersebar di 5 lokasi
- Mini map per spot
- Upgrade joran (Bambu → Fiber → Carbon → Titanium)
- Koleksi ikan
- PWA — bisa di-install ke homescreen HP

## Tech Stack
- Pure HTML5 / CSS / JavaScript
- Canvas 2D API
- Tidak butuh backend, database, atau API apapun

## Deploy
Hosted di **Cloudflare Pages** — gratis, auto-deploy dari GitHub.

## Cara Main
1. Buka peta → pilih lokasi → tap **MANCING ►**
2. Tahan tombol **LEMPAR** lalu lepaskan (power bar menentukan jarak)
3. Tunggu bobber bergerak → **TARIK!** berkali-kali sebelum senar putus
4. Bicara ke **NPC** untuk quest & info lokal
5. Cek **QUEST** board untuk misi aktif

Keyboard: `SPACE` untuk semua aksi
